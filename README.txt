Sock It To Me website package

Files:
- index.html
- styles.css
- script.js
- poster.png

How to upload to Vercel:
1. Go to vercel.com
2. Create a new project
3. Choose upload
4. Upload all files from this folder
5. Deploy

To edit:
- Change text in index.html
- Update colors/layout in styles.css
- Replace poster.png with your final poster using the same filename
